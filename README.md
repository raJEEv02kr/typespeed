## ğŸ“Š How It Works

* The user clicks **Start Test**, enabling the text area and starting a 60-second timer.
* As the user types, input is recorded.
* When the timer ends:

  * **WPM** is calculated based on words typed.
  * **Accuracy** is calculated by comparing typed characters with the given text.
* Results are displayed instantly on the screen.
